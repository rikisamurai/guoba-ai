## String Utilities

### capitalize

Capitalize the first letter of a string.

```ts
function capitalize<T extends string>(str: T): Capitalize<T>
```

```ts
capitalize('hello') // 'Hello'
```

### ensurePrefix

Ensure a string starts with a given prefix.

```ts
function ensurePrefix(str: string, prefix: string): string
```

```ts
ensurePrefix('/path', '/') // '/path'
ensurePrefix('path', '/') // '/path'
```

### ensureSuffix

Ensure a string ends with a given suffix.

```ts
function ensureSuffix(str: string, suffix: string): string
```

```ts
ensureSuffix('file.ts', '.ts') // 'file.ts'
ensureSuffix('file', '.ts') // 'file.ts'
```

### template

Simple template string replacement. Replaces `{key}` placeholders with values from the data object.

```ts
function template(str: string, data: Record<string, string>): string
```

```ts
template('Hello {name}!', { name: 'World' }) // 'Hello World!'
```

### slash

Convert backslashes to forward slashes.

```ts
function slash(str: string): string
```

```ts
slash('foo\\bar\\baz') // 'foo/bar/baz'
```
