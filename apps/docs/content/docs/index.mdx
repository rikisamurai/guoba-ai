---

## Installation

```bash
pnpm add @guoba/core
```

## Usage

```ts
import { toArray, isString, capitalize } from '@guoba/core'

toArray(1) // [1]
isString('hello') // true
capitalize('hello') // 'Hello'
```

## Features

- Pure ESM, tree-shakable
- Fully typed with TSDoc
- Zero dependencies
